Returns Policy

Most unopened products can be returned within 30 days of delivery.

Opened electronics require troubleshooting before return approval. If the
product is defective, the agent should collect the order number, product serial
number, and a short description of the issue.

Return labels can be offered for eligible returns, but refunds or credits should
not be promised until the return is approved and received.

Items marked final sale are not eligible for standard return unless they arrived
damaged or incorrect.
